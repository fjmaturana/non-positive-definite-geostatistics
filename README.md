# Non-Positive-Definite Covariance Structures in Geostatistics

Code accompanying:

Maturana, F.J.
"Non-Positive-Definite Covariance Structures in Geostatistics:
Theory, Diagnostics, and Implications for Kriging"

## Contents

- numerical_simulation.py
  Synthetic experiments.

- kimberley_pickand_case_study.py
  Kimberley stream-sediment case study.

## Requirements

Python 3.11+

numpy
scipy
pandas

## Reproducing the paper

Run:

python numerical_simulation.py

and

python kimberley_pickand_case_study.py
